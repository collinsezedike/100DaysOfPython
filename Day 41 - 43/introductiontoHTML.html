There are two kinds of list:
Ordered list and unordered list
The ol tag creates an ordered list

  2. i

  4. e
  6. t

  9. s
The ul tag is used to create unordered lists

The li tag is used to add add an item to the list. Mere ol or ul tags do nothing.

To add table to a web page, use the table tag.
Like this
thead To declare the header of the table
tbody To declare the body of the table
tfoot To declare the footer of the table
tr To add a table row
th To add a table header item
td To add a table body item
By default, tables are styleless The border attribute is used to add table borders


The form tag is used to create a form element
However, just like the table tag, it is useless without two other tags.
The label tag and the input tag
The label tag is used to give a label name to the form field
The input tag requires a type attribute that determines the type of form field to be displayed.
